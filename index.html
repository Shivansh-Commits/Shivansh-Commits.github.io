Context Forge

Schema-agnostic Java library that converts any hierarchical JSON specification into LLM-friendly plain text via Structured Specification Injection (SSI). Produces a structure tree (node ordering, cardinality, group boundaries) and per-node detail specs — injected as Tier 1 normative context in a RAG pipeline to eliminate specification hallucination. Works with EDIFACT messages, REST API specs, database schemas, and any domain-specific JSON.
